ℹ️ About This HTTP Server

This server is a complete implementation of HTTP/1.1 protocol built from scratch using Java's socket library.

✨ Key Features

  • 🔌 Raw Socket Programming - No frameworks, built from TCP sockets
  • 🧵 Thread Pool - Handles multiple concurrent connections efficiently
  • 📦 Binary File Transfer - Serves images, documents with integrity
  • 📮 POST Request Handling - Accepts and processes JSON data
  • 🔒 Security Features - Path traversal protection, host validation
  • 🔄 Keep-Alive Support - Persistent connections for better performance
  • ⏱️ Connection Timeouts - Automatic cleanup of idle connections
  • 📊 Comprehensive Logging - Detailed request/response tracking

🛠️ Technology Stack

Java 17 Socket API ThreadPoolExecutor HTTP/1.1 JSON

📖 Learning Objectives

This project demonstrates understanding of:

  • Network protocols and the OSI model
  • TCP/IP communication
  • HTTP request/response cycle
  • Concurrent programming patterns
  • File I/O and binary data handling
  • Web security fundamentals
